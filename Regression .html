Understanding Logistic Regression through Linear Regression:

Linear regression is very straight forward and logical to understand. As we tweak X, Y will come out. Can it be done to categorical responses (Classification)?
No, It can not be done. There are mainly two reasons a.) Regression method is not bounded in probability outcome i.e Y is not be (0,1) b.) Regression output is not designed to output multinomial value.
But, we can derive mathematically to achieve from linear regression to logistic regression to bound the outcome (0,1), probability to find the output.
